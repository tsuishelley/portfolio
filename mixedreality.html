Developing signage and brand material for Mixed Reality Developer Program.

Background

This was a project done for Microsoft's Hololens in which we developed a visual identity for both their program and their dev days hackathon. Using a color palette inspired by the command line interface, we developed a brand system composed of animated spirographs, futuristic gradients, and the bold Lineto type face which pays homage to early game consoles.
